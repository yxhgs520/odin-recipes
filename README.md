# Odin Recipes

This is a recipe website project created as part of **The Odin Project** curriculum. The project focuses on practicing basic HTML skills, including links, images, and directory structures.

## Demo
The project consists of a main index page linked to various recipe pages.

## Technologies Used
* HTML5

## Project Structure
- `index.html`: The main landing page.
- `recipes/`: A directory containing detailed recipe pages (e.g., [boiledfish.html](https://github.com/yxhgs520/odin-recipes/tree/main/recipes)).
- `pictures/`: A folder for storing all food images used in the project.
